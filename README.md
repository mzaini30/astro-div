# astro-div

It's only `<Div/>`

## Example

```astro
---
import Div from 'astro-div'
---

<!DOCTYPE html>
<html lang="en">
<head>
	<Div>
		<meta charset="UTF-8" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<title>Document</title>
	</Div>
</head>
<body>
	
</body>
</html>
```

Result:

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body></body>

</html>
```
